# Patient Record ID Management System 🏥

**Developed:** Data Structures and Algorithms Project

A high-performance C++ application designed to manage large datasets of unique patient identifiers. This project implements advanced non-linear data structures to ensure optimal search, insertion, and deletion times for critical health records.

## ✨ Features

* **Self-Balancing Architecture:** Implemented a custom AVL Tree from scratch to manage patient IDs, strictly maintaining height balance to guarantee `O(log N)` time complexity for all core operations.
* **Algorithmic Structural Manipulation:** Engineered over 15 distinct recursive functions to handle advanced structural tasks, including patient ranking, in-order traversals, and dynamic database merging.
* **Automated Rotations:** Developed robust logic for Single (Left/Right) and Double (Left-Right/Right-Left) rotations to automatically self-correct tree skewness during live data insertion.
* **Skew Detection Analysis:** Includes built-in analytical tools to evaluate and display current tree balance states and node distribution metrics.

## 🛠️ Tech Stack

* **Language:** C++
* **Concepts:** Advanced Data Structures, Object-Oriented Programming (OOP), Recursive Algorithms, Memory Management
* **Environment:** Visual Studio 2022
